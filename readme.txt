== Description ==

This plugin adds a column to the plugin's page in WordPress's admin to show each plugin rating.
[Plugin Page](https://virtumente.com/wp/plugins/plugins-rating-column/ "Plugins_Ratings_Column").  


== Installation ==

1. Install this plugin either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it. You're ready to go!
